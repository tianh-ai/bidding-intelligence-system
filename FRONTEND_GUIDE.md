## 🚀 快速启动

### 方式 1：一键启动（推荐）

```bash
cd frontend
./start.sh
```

### 方式 2：手动启动

```bash
cd frontend

# 1. 安装依赖（首次）
npm install

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 设置后端地址（默认 http://localhost:8000）

# 3. 启动
npm run dev
```

### 访问地址

- **前端**: http://localhost:3000
- **默认账号**: `admin` / `admin123`

## 💡 核心页面功能

### 1. 逻辑学习页面（最复杂）

完整工作流：

```
1. 选择文件/文件夹
2. 点击"开始学习" → 显示进度
3. 学习完成 → 显示逻辑库（永久 + 临时）
4. 点击"生成投标文件"
5. 自动检查 → 右侧显示检查结果
6. 人工添加反馈
7. 点击"应用反馈并重新生成"
8. AI 学习修改逻辑 → 再次生成
9. 满意后点击"保存逻辑到逻辑库"
```

### 2. 文件上传页面

- 拖拽上传
- 批量上传
- 自动匹配招标/投标文件
- 显示匹配结果

### 3. 大模型管理页面

- 模型列表
- 添加/编辑/删除模型
- API Key 输入（前端管理）
- 模型测试功能
