第一部分 投标须知

一、项目概况
1.1 项目名称
本项目为智能办公系统建设项目。

1.2 项目地点
项目位于北京市海淀区中关村软件园。

二、投标人资格要求
2.1 基本资格
投标人应具备以下资格：
(1) 在中华人民共和国境内注册，具有独立法人资格。
(2) 具有有效的营业执照、税务登记证和组织机构代码证。

2.2 技术资格
投标人应具备软件开发能力和系统集成资质。

第二部分 技术要求

一、系统架构
1.1 总体架构
系统采用微服务架构，包含以下模块：
- 用户管理模块
- 文档管理模块  
- 工作流模块

1.2 技术栈
后端：Python FastAPI
前端：React + TypeScript
数据库：PostgreSQL

二、性能要求
2.1 响应时间
系统响应时间不超过2秒。

2.2 并发用户
支持1000个并发用户。

第三部分 商务要求

一、报价要求
1.1 报价构成
报价应包含软件开发费、系统集成费、培训费等。

1.2 付款方式
按照合同进度分期付款。
